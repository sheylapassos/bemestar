# Bem Estar — App Flutter

Este é um aplicativo simples de bem-estar desenvolvido com Flutter, como parte da atividade de sistematização da disciplina.

Esta atividade propõe a criação de um aplicativo alinhado aos Objetivos de Desenvolvimento Sustentável (ODS) da ONU, voltado ao ODS 3 — Saúde e Bem-Estar, com foco em idosos, visando promover autonomia, segurança e qualidade de vida por meio de lembretes de medicação, metas semanais de hábitos saudáveis e suporte 

## Funcionalidades

- Tela de boas-vindas (Início)
- Cadastro de medicações com nome, dosagem e horário
- Lista dinâmica de medicações com opção de remoção
- Tela de hábitos saudáveis
