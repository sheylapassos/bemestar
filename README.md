# Bem Estar — App Flutter

Este é um aplicativo simples de bem-estar desenvolvido com Flutter, como parte da atividade de sistematização da disciplina.

## Funcionalidades

- Tela de boas-vindas (Início)
- Cadastro de medicações com nome, dosagem e horário
- Lista dinâmica de medicações com opção de remoção
- Tela de hábitos saudáveis

## Como rodar o projeto

1. Instale o Flutter: https://docs.flutter.dev/get-started/install
2. Clone o repositório: git clone https://github.com/sheylapassos/bemestar.git
